# GASTLI

Open-source gas-giant interior structure code (Acuña et al. 2021, 2024, 2025; not yet ingested; github.com/lorenaacuna/GASTLI). Computes a 2-layer planet (rocky core + H/He+metals envelope) on a hot-start adiabat down to P = 1000 bar. Designed to couple with [[petitRADTRANS]] atmospheric retrievals in a [[joint-interior-atmosphere-retrieval]] framework, sampling core mass fraction and envelope metallicity simultaneously with atmospheric parameters.

## Papers

- [[2025_AcunaAguirre_WASP80b]] — primary interior code in the WASP-80 b joint interior–atmosphere retrieval; CMF and Z_env sampled jointly with atmospheric M/H, C/O, and clouds.
