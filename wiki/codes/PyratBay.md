# PyratBay

PyratBay (Cubillos & Blecic 2021; not yet ingested) — open-source Python radiative-transfer and Bayesian atmospheric retrieval framework for transmission and emission spectra. Supports free chemistry, equilibrium chemistry (via TEA), parametric T–P profiles, cloud / haze parameterizations.

## Papers

- [[2026_Radica_WASP96b]] — one of four free-chemistry retrieval frameworks cross-compared on WASP-96 b (alongside [[POSEIDON]], [[NEMESISPY]], [[Aurora]]); flagged as the only retrieval to find tentative CH₄ on WASP-96 b — likely an artifact.
