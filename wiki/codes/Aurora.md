# Aurora

Aurora (Welbanks & Madhusudhan 2021; not yet ingested) — Bayesian transmission-spectrum retrieval code with multidimensional inhomogeneous-cloud / haze treatments and limb-asymmetry support. Designed for hot-Jupiter / sub-Saturn atmospheres.
