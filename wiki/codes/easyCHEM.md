# easyCHEM

Chemical equilibrium code (Mollière et al. 2017; not yet ingested) commonly used as the equilibrium-chemistry module for [[petitRADTRANS]] forward models and retrievals. An alternative to [[FastChem]] for equilibrium abundance calculations across atmospheric T–P grids.

## Papers

- [[2025_AcunaAguirre_WASP80b]] — equilibrium-chemistry module for the joint interior–atmosphere retrieval on WASP-80 b within the petitRADTRANS framework.
- [[2025_Gordon_COMPASS-G395H]] — used with [[PICASO]] for atmospheric grid construction (this work) cross-checked against [[PICASO]] + [[Photochem]] grids reproducing prior COMPASS analyses.
