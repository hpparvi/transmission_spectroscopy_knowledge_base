# WASP-96

Late-G dwarf hosting the inflated hot Saturn [[WASP-96b]]. Stellar C/O = 0.42 (Nikolov et al. 2022; not yet ingested) is a key reference value for interpreting the planet's near-stellar retrieved C/O.
